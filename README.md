# ETL de Séries (TV Shows ETL)

Este projeto é um pipeline **ETL (Extract, Transform, Load)** desenvolvido em Python, criado para o Desafio da DIO.  
Ele processa uma lista de séries, aplica transformações úteis e carrega tudo em um banco de dados local SQLite.

O objetivo deste projeto é demonstrar como construir um ETL simples, modular e organizado, usando apenas Python, Pandas e SQLite.
